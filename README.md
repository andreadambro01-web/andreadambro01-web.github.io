# Portfolio HTML, CSS & SCSS - Andrea D'Ambrosio

Questo è il mio sito personale realizzato per il corso Start2Impact.  
Il progetto è stato sviluppato con attenzione alla struttura, alla responsività e all’organizzazione del codice tramite SCSS.

## Struttura del progetto

- `index.html` → Home page con presentazione personale e progetti
- `cv.html` → Curriculum Vitae in formato HTML
- `contatti.html` → Pagina contatti con form di invio messaggi
- `src/scss/` → Codice sorgente SCSS organizzato con pattern 7-1
- `dist/css/style.css` → File CSS compilato da SCSS
- `assets/images/` → Immagini personali e progetti
- `assets/favicons/` → Favicon del sito

## Tecnologie utilizzate

- HTML5
- CSS3 / SCSS
- Bootstrap 5
- Flexbox e CSS Grid
- Git / GitHub
- Google Fonts
- EmailJS (per invio form contatti)

## Funzionalità

- Layout completamente responsive (mobile-first)
- Menu di navigazione sticky con versione mobile a collapse
- Struttura SCSS modulare (7-1 pattern)
- Utilizzo di variabili, mixin e funzioni SCSS
- Form di contatto funzionante con invio email tramite EmailJS
- Utilizzo di Open Graph meta tags per condivisione social
- Favicon personalizzata

## Obiettivi del progetto

Questo progetto ha lo scopo di dimostrare la capacità di:
- strutturare un sito multi-pagina
- organizzare codice CSS tramite SCSS
- utilizzare framework frontend (Bootstrap)
- realizzare layout responsive moderni
- gestire componenti riutilizzabili e mantenibili

## Note

- Il sito è completamente responsive su tutti i dispositivi
- Il menu si trasforma in hamburger button su mobile
- SCSS è organizzato per migliorare scalabilità e manutenzione
- Il progetto è pubblicato tramite GitHub Pages
