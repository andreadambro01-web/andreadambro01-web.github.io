# Portfolio HTML, CSS & SCSS - Andrea D'Ambrosio

Questo è il mio sito personale creato per il corso Start2Impact. Ora il progetto include SCSS, Google Fonts e un menu responsive con button mobile.

## Contenuto
- `index.html` → Home page con bio e progetti
- `cv.html` → Curriculum Vitae in formato HTML
- `contatti.html` → Pagina con form di contatto
- `src/scss/` → Codice sorgente SCSS
- `dist/css/style.css` → CSS compilato dallo SCSS
- `img/` → Immagini personali e dei progetti
- Favicon inclusa

## Tecnologie
- HTML
- CSS / SCSS
- Bootstrap 5
- Flexbox / Grid
- Git / GitHub
- Google Fonts

## Note
- Il sito è completamente responsive.
- Il menu diventa un button per la visualizzazione mobile.
- Ho usato SCSS per strutturare meglio i fogli di stile e poter gestire facilmente colori, font e componenti.
